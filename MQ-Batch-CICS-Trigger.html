2. PUT via Job Batch

2.1 Programa COBOL: MQPUT01B

O programa conecta ao QMGR CSQ7, abre a fila informada via PARM e envia a mensagem lida do cartão SYSIN.

O fonte completo pode ser acessado aqui MQPUT01B

IDENTIFICATION DIVISION. PROGRAM-ID. MQPUT01B. DATA DIVISION. WORKING-STORAGE SECTION. 01 MQ-CONSTANTS. COPY CMQV. COPY CMQODV. COPY CMQMDV. COPY CMQPMOV. 01 WS-QMGR-NAME PIC X(48) VALUE 'CSQ7'. 01 WS-QUEUE-NAME-TEXT PIC X(48) VALUE SPACES. 01 WS-MSG-LEN PIC S9(9) COMP VALUE 100. 01 WS-HCONN PIC S9(9) COMP VALUE 0. 01 WS-HOBJ PIC S9(9) COMP VALUE 0. 01 WS-COMP-CODE PIC S9(9) COMP VALUE 0. 01 WS-REASON-CODE PIC S9(9) COMP VALUE 0.

2.2 JCL para Execução

//JMQPUT01 JOB ('00'),TSO.&SYSUID,NOTIFY=&SYSUID, // MSGCLASS=T,MSGLEVEL=1,CLASS=A //*-------------------------------------------------------------* //* COLOCAR MENSAGEM NA FILA * //*-------------------------------------------------------------* //STEP1 EXEC PGM=MQPUT01B, // PARM='LAB.LOCAL.APP.REQ.CICS' //STEPLIB DD DISP=SHR,DSN=USER.LOAD // DD DISP=SHR,DSN=CSQ701.SCSQLOAD // DD DISP=SHR,DSN=CSQ701.SCSQAUTH // DD DISP=SHR,DSN=CSQ701.SCSQANLE //SYSPRINT DD SYSOUT=* //SYSIN DD * MENSAGEM PARA O JOB BATCH /*

5. GET via Transação CICS

5.1 Definição dos Recursos Transação TEST e programa MQGET02C no CICS

CEDA DEF TRANSaction(TEST) GROUP(GAOR) PROGram(MQGET02C)
CEDA View TRANSaction( TEST ) TRANSaction : TEST Group : GAOR DEScription ==> PROGram ==> MQGET02C

CEDA DEFINE PROGram(MQGET02C) GROUP(GAOR) Language(CObol)
CEDA View PROGram( MQGET02C ) PROGram : MQGET02C Group : GAOR DEScription : Language : CObol

5.2 Recursos Instalados

CEMT INQUIRE TRANSaction(TEST) Tra(TEST) Pri( 001 ) Pro(MQGET02C) Tcl( DFHTCL00 ) Ena Sta CEMT INQUIRE PROGram(MQGET02C) Prog(MQGET02C) Leng(0000000000) Cob Pro Ena Pri

5.3 Conexão com o MQ

CEMT INQUIRE MQCONN Mqname( CSQ7 ) Mqqmgr(CSQ7) Connectst( Connected )

6. Trigger MQ → CICS

Configurando um trigger no MQ para executar transação CICS.


O fonte completo do programa MQTRIGON pode ser acessado aqui.
IDENTIFICATION DIVISION. PROGRAM-ID. MQTRIGON. DATA DIVISION. WORKING-STORAGE SECTION. 01 MQ-OD-AREA. COPY CMQODV. 01 MQ-MD-AREA. COPY CMQMDV. 01 MQ-GMO-AREA. COPY CMQGMOV. 01 MQ-CONST-AREA. COPY CMQV. 01 WS-QUEUE-NAME-TEXT PIC X(48) VALUE SPACES. 01 WS-MSG-LEN PIC S9(9) COMP VALUE 100. 01 WS-MSG-DATA-LEN PIC S9(9) COMP VALUE 0. 01 WS-MSG-TEXT PIC X(100) VALUE SPACES. 01 WS-HCONN PIC S9(9) COMP VALUE 0. 01 WS-HOBJ PIC S9(9) COMP VALUE 0. 01 WS-COMP-CODE PIC S9(9) COMP VALUE 0. 01 WS-REASON-CODE PIC S9(9) COMP VALUE 0. 01 WS-MSG-SAIDA PIC X(120) VALUE SPACES. 01 WS-COMP-CODE-DISP PIC Z(8)9. 01 WS-REASON-CODE-DISP PIC Z(8)9. 01 WS-RESP-DISP PIC Z(8)9. 01 WS-TS-QUEUE PIC X(8) VALUE 'TEST '. 01 WS-TS-ITEM PIC S9(4) COMP VALUE 1. 01 WS-TS-RECORD PIC X(120) VALUE SPACES. 01 WS-TS-RESP PIC S9(8) COMP VALUE 0. 01 WS-TERM-MSG PIC X(72) VALUE SPACES. 01 WS-OPER-MSG PIC X(80) VALUE SPACES.

6.1 PROCESS no MQ

DEFINE PROCESS(LAB.PROCESS.CICS.TRIG) - APPLTYPE(CICS) - APPLICID('TRIG') - DESCR('ACIONA TRANSACAO TRIG NO CICS') - REPLACE

6.2 Trigger na Fila

ALTER QLOCAL(LAB.LOCAL.APP.REQ.CICS) - TRIGGER(ON) - TRIGTYPE(FIRST) - TRIGDPTH(1) - INITQ(CSQ7.TRIGGER.CICSA02) - PROCESS(LAB.PROCESS.CICS.TRIG)

6.3 Recursos CICS

CEDA VIEW TRANSaction(TRIG) TRANSaction : TRIG PROGram : MQTRIGON CEDA VIEW MQconn(CSQ7) Initqname : CSQ7.TRIGGER.CICSA02

6.4 Disparando o Trigger

CEMT INQUIRE MQCONN Tasks(0001) Trigmontasks(0001) ← Trigger processado!

Podemos visualizar na TS TEST a mensagem.
